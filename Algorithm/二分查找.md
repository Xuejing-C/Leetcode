# 二分查找
## 一、知识点
### 1. 前提
有序数组；数组中无重复元素。
## 二、题目
```Java
class Solution {
    public int search(int[] nums, int target) {
        int left = 0;
        int right = nums.length - 1;
        int mid;
        if(target < nums[left] || target > nums[right]){
            return -1;
        }
            
        while(left <= right){
            mid = (left + right) / 2;
            if(target > nums[mid]){
                left = mid + 1;
            }else if(target < nums[mid]){
                right = mid - 1;
            }else{
                return mid;
            }
        }
        return -1;
    }
}
```
